# Edify Project Structure

The old root prototype files were removed so the project is not confusing.

Two current versions remain:

```text
Web/
  Browser/Vite version of Edify with the same interface.

App/
  Electron desktop version of Edify that builds a Windows .exe installer.
```

Use `Web` when you want to run or publish the site-style preview.

Use `App` when you want the real Windows desktop application.
