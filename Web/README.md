# Edify Web

This folder keeps the web/preview version of Edify separate from the Windows desktop app.

It uses the same React interface and demo project, but it runs with browser/localStorage fallbacks instead of native Electron file dialogs.

## Run

```powershell
npm.cmd install
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
```

## Preview Built Site

```powershell
npm.cmd run preview
```
